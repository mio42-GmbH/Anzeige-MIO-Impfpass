# Prototyp des neuen MIO Viewer für das MIO Impfpass

<br />
In den ersten Iterationen unseres Projektes wird ein Prototyp für die Anzeige des **MIO Impfpass
1.1.0** entwickelt. Künftig sollen Viewer für alle MIOs folgen. Der Prototyp umfasst aktuell
eine interaktive JavaScript Webkomponente sowie eine Bibliothek von Unterkomponenten entwickelt
mit StencilJS. Die übergreifende Webkomponente kann als Blackbox genutzt werden, welche MIO-Daten
als statischen Base64-String entgegennimmt. Für das Einlesen der MIO-Daten wird intern der
MIO Parser genutzt. Für mehr Flexibilität, können die Subkomponenten ebenfalls separat als
Webkomponenten genutzt werden.

Den aktuellen Stand des Prototypen finden Sie in unserem [Github Repository](https://github.com/mio42-GmbH/Anzeige-MIO-Impfpass)

<br />
